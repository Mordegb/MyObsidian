#### <span style="color:rgb(0, 230, 172)">$git init</span> -> inicia um repositório git em sua maquina local, já permitindo o acesso a controle de versionamento

#### <span style="color:rgb(0, 230, 172)">$git add</span> <span style="color:rgb(255, 255, 0)">nome_do_arquivo</span> ->  apos fazer a mudanças desejadas deixa o arquivo pronto para commitar

#### <span style="color:rgb(0, 230, 172)">$git commit -m </span>"<span style="color:rgb(255, 255, 0)">mensagem_resumo</span>" -> salva os arquivos modificados e o usuário bota uma mensagem com o resumo das modificações 

#### <span style="color:rgb(0, 230, 172)">$git branch</span> -> fala sua branch atual 

#### <span style="color:rgb(0, 230, 172)">$</span><span style="color:rgb(0, 230, 172)">git push origin</span> <span style="color:rgb(255, 255, 0)">nome_da_branch</span><span style="color:rgb(255, 255, 0)"> </span> -> salva os arquivos commitados e sobe eles para o seu repositório remoto ou em seu versionamento do código local , tudo em sua respectiva Branch

#### <span style="color:rgb(0, 230, 172)">$git pull origin</span> <span style="color:rgb(255, 255, 0)">nome_da_branch</span> -> atualiza seu repositório local com as mudanças feitas por outras pessoas puxando do repositório remoto,isso referente a branch que você está

#### <span style="color:rgb(0, 230, 172)">$git switch</span> <span style="color:rgb(255, 255, 0)">nome_da_branch</span> -> sai da sua branch atual e vai pra que você colocou o nome (a outra branch tem que existir)

#### <span style="color:rgb(0, 230, 172)">$git chekout -b</span> <span style="color:rgb(255, 255, 0)">nome_da_branch</span> -> sai da sua branch atual e vai para uma nova branch com o nome que você colocou, a nova branch também terá os mesmos arquivos da branch que você saiu

#### <span style="color:rgb(0, 230, 172)">$git status</span> -> fala oque falta ser commitado e oque já está

#### <span style="color:rgb(0, 230, 172)">$git fetch --all </span> ->  puxa todas as branchs e informações do repositório remoto 

#### <span style="color:rgb(0, 230, 172)">$git log</span> -> mostra o histórico de commit's


Dicas importantes:
	sempre dé os comando "git fetch --all" e "git pull" quando entrar numa branch,pra não dar merda. sempre faça o merge e o pull request pelo próprio github. quando criar um repositório no github so leia e siga os paços que tem la, vai facilitar
